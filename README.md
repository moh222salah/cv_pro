## ğŸš€ Quick Start

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server required - runs entirely in the browser

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/cv-generator.git
   cd cv-generator
   ```

2. **Open in browser**
   ```bash
   # Simply open index.html in your browser
   open index.html  # macOS
   start index.html  # Windows
   xdg-open index.html  # Linux
   ```

3. **Or use a local server** (recommended)
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve
   
   # Using PHP
   php -S localhost:8000
   ```

4. **Visit** `http://localhost:8000`

## ğŸ“– Usage Guide

### 1. **Fill Personal Information**
- Enter your full name, job title, email, and phone
- Add location and LinkedIn profile (optional)
- Write a compelling professional summary

### 2. **Add Experience**
- Click "Add Experience" to add multiple positions
- Include position, company, duration, and location
- List key responsibilities (one per line)

### 3. **Add Education**
- Add your degrees and certifications
- Include institution, year, and grade/GPA
- Can add multiple education entries

### 4. **List Skills**
- Enter comma-separated technical skills
- Skills will appear as professional tags

### 5. **Add Languages**
- Specify languages you speak
- Select proficiency level for each

### 6. **Upload Photo (Optional)**
- Check "Include photo in CV"
- Upload a professional headshot
- Preview before finalizing

### 7. **Preview & Download**
- Click "Preview" to see your CV
- Choose English or Arabic version
- Click "Download" to get PDF
